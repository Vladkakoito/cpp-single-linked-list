Односвязный список

Тренировка работы с динамической памятью. Каждый элемент содержит значение и указатель на следующий элемент.

Память освобождается в деструкторе и методом Clear.
Поддерживаются методы swap, PushFront, GetSize, IsEmpty, EraseAfter, PopFront, InsertAfter, и методы работы с итераторами.
